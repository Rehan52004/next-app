<h2>Steps to run the Application</h2>

<h4>How to run Backend</h4>

```javascript
  cd blog-backend
  npm i
  npm start
```

<h4>How to run Frontend</h4>

```javascript
  cd my-next-app
  npm i
  npm run dev
```
